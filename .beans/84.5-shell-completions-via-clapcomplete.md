## Problem

35+ subcommands with no tab completion. For a CLI tool, this is a major DX gap.

## Approach

Wire up `clap_complete` to generate completions for bash, zsh, fish, and PowerShell.

### Files to modify
- `Cargo.toml` — add `clap_complete` dependency
- `src/cli.rs` — add `completions` subcommand
- `src/main.rs` — handle `Command::Completions`

### Design
- `bn completions bash` prints completion script to stdout
- User adds `eval \"$(bn completions bash)\"` to their shell rc
- Follow clap_complete standard pattern

### Acceptance Criteria
- [ ] `bn completions bash` outputs valid bash completion script
- [ ] `bn completions zsh` outputs valid zsh completion script
- [ ] `bn completions fish` outputs valid fish completion script
- [ ] Completions include all subcommands and flags
