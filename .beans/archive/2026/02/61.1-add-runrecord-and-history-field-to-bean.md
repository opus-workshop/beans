  ## Task
  Add RunRecord struct and history field to Bean for structured run tracking.

  ## Steps
  1. Define RunResult enum in src/bean.rs:
  ```rust
  #[derive(Debug, Clone, PartialEq, Serialize, Deserialize)]
  #[serde(rename_all = "snake_case")]
  pub enum RunResult {
      Pass,
      Fail,
      Timeout,
      Cancelled,
  }
  ```

  2. Define RunRecord struct in src/bean.rs:
  ```rust
  #[derive(Debug, Clone, PartialEq, Serialize, Deserialize)]
  pub struct RunRecord {
      pub attempt: u32,
      pub started_at: DateTime<Utc>,
      #[serde(skip_serializing_if = "Option::is_none")]
      pub finished_at: Option<DateTime<Utc>>,
      #[serde(skip_serializing_if = "Option::is_none")]
      pub duration_secs: Option<f64>,
      #[serde(skip_serializing_if = "Option::is_none")]
      pub agent: Option<String>,
      pub result: RunResult,
      #[serde(skip_serializing_if = "Option::is_none")]
      pub exit_code: Option<i32>,
      #[serde(skip_serializing_if = "Option::is_none")]
      pub tokens: Option<u64>,
      #[serde(skip_serializing_if = "Option::is_none")]
      pub cost: Option<f64>,
      #[serde(skip_serializing_if = "Option::is_none")]
      pub output_snippet: Option<String>,
  }
  ```

  3. Add history field to Bean struct:
  ```rust
  #[serde(default, skip_serializing_if = "Vec::is_empty")]
  pub history: Vec<RunRecord>,
  ```

  4. Add `history: Vec::new()` to Bean::try_new defaults
