## Task
Display run history in `bn show` output when history is non-empty.

## Steps
1. In src/commands/show.rs, after displaying existing fields, add a History section:
```
## History
  #  Result   Duration  Agent        Exit  Tokens
  1  fail     12.3s     pi-abc123    1     8.2k
  2  fail     8.1s      pi-def456    1     6.1k  
  3  pass     15.7s     pi-ghi789    0     12.4k
```

2. Format rules:
   - Skip section entirely if history is empty
   - Duration: format as Xs, Xm Xs, or Xh Xm
   - Tokens: format with k suffix (e.g., 12.4k)
   - Cost: format as $X.XX if present
   - Agent: truncate to 12 chars if longer
   - Show most recent N entries (default 10, all with --history flag)

3. Add totals line at bottom:
```
  Total: 3 attempts, 36.1s, 26.7k tokens, $0.12
```
