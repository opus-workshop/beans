New module src/agent_presets.rs with:

1. Agent preset struct with name, run template, plan template, version command
2. Built-in presets for: pi, claude, aider
3. `detect_agents()` — scan PATH for known agent CLIs, return which are available
4. `get_preset(name)` — get a preset by name
5. `all_presets()` — list all known presets

## Presets

| Agent | run | plan | version_cmd |
|-------|-----|------|-------------|
| pi | `pi @.beans/{id}-*.md "implement and bn close {id}"` | `pi @.beans/{id}-*.md "plan into children with bn create --parent {id}"` | `pi --version` |
| claude | `claude -p "implement bean {id} and run bn close {id}"` | `claude -p "bean {id} is too large, split with bn create --parent {id}"` | `claude --version` |
| aider | `aider --message "implement bean {id}, verify with bn close {id}"` | `aider --message "plan bean {id} into children with bn create"` | `aider --version` |
