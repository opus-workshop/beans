## Task
Merge the deli extension (parallel agent TUI) and beans extension (status widget)
into a single unified beans extension. Change all deli binary calls to bn calls.

## Changes

### 1. Merge into ~/.pi/agent/extensions/beans/index.ts
Take all tools from deli extension and add them to beans extension:
- deli_run → beans_run (calls `bn run --json-stream` instead of `deli --json-stream run`)
- deli_spawn → beans_spawn (calls `bn run --json-stream <id>` instead of `deli --json-stream spawn`)
- deli_pull → beans_pull (calls `bn run --json-stream <parent>` with fire-and-forget)
- deli_status → beans_status (calls `bn agents` instead of `deli status`)
- deli_logs → beans_logs (calls `bn logs` instead of `deli logs`)

### 2. Keep all TUI components
The DeliProgressComponent, spinner, progress bar — all stay. Just rename
references from 'deli' to 'beans' in labels and widget IDs.

### 3. Keep existing beans extension features
Status widget, /beans command, Ctrl+B shortcut — all stay.

### 4. Remove deli extension
After merging, the deli extension directory can be deleted.

## Files
- ~/.pi/agent/extensions/beans/index.ts (major rewrite — merge deli into this)
- ~/.pi/agent/extensions/deli/ (delete after merge)

## Key change in spawn calls
Before: `spawn("deli", ["--json-stream", "run", parentId, ...])`
After:  `spawn("bn", ["run", parentId, "--json-stream", ...])`

Before: `spawn("deli", ["--json-stream", "spawn", beanId, ...])`  
After:  `spawn("bn", ["run", beanId, "--json-stream", ...])`

Before: `spawn("deli", ["status"])`
After:  `spawn("bn", ["agents", "--json"])`
