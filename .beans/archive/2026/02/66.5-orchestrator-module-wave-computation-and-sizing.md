New module src/orchestrator.rs — wave computation, bean sizing, and dispatch logic.

This is pure logic (no process spawning). Uses existing beans internals directly
(not shelling out to bn CLI like deli/bw did).

## Functions

1. `get_ready_beans(beans_dir) -> Vec<Bean>`
   Use existing index + ready logic from src/commands/ready.rs.
   Filter to open beans with no unmet dependencies.

2. `size_bean(bean, beans_dir) -> SizedBean`
   Read bean content + file context (via ctx_assembler).
   Estimate tokens (len/4 heuristic, same as existing tokens.rs).
   Classify: Implement (below max_tokens) or Plan (above).

3. `compute_waves(beans: &[Bean]) -> Vec<Vec<Bean>>`
   Port from deli/src/beans.rs. Group beans by dependency order.
   Wave 0: no deps. Wave 1: deps all in wave 0. Etc.
   Handle cycles gracefully (warn and skip).

4. `plan_dispatch(beans_dir, config) -> DispatchPlan`
   Combines the above: get ready beans, size them, compute waves,
   return a plan struct with:
   - waves: Vec<Wave> where Wave has implement_beans and plan_beans
   - skipped: Vec<Bean> (large beans when not auto-plan)

## Types

```rust
pub enum BeanAction { Implement, Plan }
pub struct SizedBean { pub bean: Bean, pub tokens: u64, pub action: BeanAction }
pub struct Wave { pub beans: Vec<SizedBean> }
pub struct DispatchPlan {
    pub waves: Vec<Wave>,
    pub skipped: Vec<SizedBean>,  // large beans needing bn plan
}
```

## Context

### Wave computation from deli (port this logic):
```rust
pub fn compute_waves(beans: &[Bean]) -> Vec<Vec<Bean>> {
    let mut waves = Vec::new();
    let mut remaining = beans.iter().filter(|b| b.status == "open").cloned().collect();
    let mut completed: HashSet<String> = beans.iter()
        .filter(|b| b.status == "closed").map(|b| b.id.clone()).collect();
    while !remaining.is_empty() {
        let (ready, blocked) = remaining.into_iter()
            .partition(|b| b.dependencies.iter().all(|d| completed.contains(d)));
        if ready.is_empty() { break; }
        for b in &ready { completed.insert(b.id.clone()); }
        waves.push(ready);
        remaining = blocked;
    }
    waves
}
```
