  Create a pi extension to extract transcripts from YouTube videos.

  ## Context
  Users may share YouTube links (tutorials, talks, meetings) and want summaries or analysis.
  YouTube provides auto-generated and manual captions that can be extracted.

  ## Task
  1. Create extension at ~/.pi/agent/extensions/youtube/index.ts
  2. Register a 'youtube_transcript' tool
  3. Accept YouTube URL or video ID
  4. Extract available transcript/captions
  5. Return formatted text with timestamps (optional)

  ## Implementation Options
  - Use youtube-transcript npm package
  - Or parse YouTube's timedtext API directly
  - Or use yt-dlp --write-sub --skip-download

  ## Files
  - ~/.pi/agent/extensions/youtube/index.ts
  - ~/.pi/agent/extensions/youtube/package.json (if using npm deps)
