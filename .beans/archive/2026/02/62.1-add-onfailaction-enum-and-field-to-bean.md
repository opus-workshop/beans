  ## Task
  Add OnFailAction enum and on_fail field to Bean struct.

  ## Steps
  1. Define OnFailAction enum in src/bean.rs:
  ```rust
  #[derive(Debug, Clone, PartialEq, Serialize, Deserialize)]
  #[serde(tag = "action", rename_all = "snake_case")]
  pub enum OnFailAction {
      /// Retry with optional max attempts and delay
      Retry {
          #[serde(skip_serializing_if = "Option::is_none")]
          max: Option<u32>,
          #[serde(skip_serializing_if = "Option::is_none")]
          delay_secs: Option<u64>,
      },
      /// Bump priority and add message
      Escalate {
          #[serde(skip_serializing_if = "Option::is_none")]
          priority: Option<u8>,
          #[serde(skip_serializing_if = "Option::is_none")]
          message: Option<String>,
      },
  }
  ```

  Note: Skip Run and Goto variants for now — they require cross-bean operations.
  Retry and Escalate are self-contained and cover the most common cases.

  2. Add on_fail field to Bean:
  ```rust
  #[serde(default, skip_serializing_if = "Option::is_none")]
  pub on_fail: Option<OnFailAction>,
  ```

  3. Add `on_fail: None` to Bean::try_new defaults

  4. Add --on-fail CLI flag to create command in src/cli.rs:
     - `--on-fail retry` → Retry { max: None, delay_secs: None }
     - `--on-fail retry:5` → Retry { max: Some(5), delay_secs: None }
     - `--on-fail escalate` → Escalate { priority: None, message: None }
     - `--on-fail escalate:P0` → Escalate { priority: Some(0), message: None }
