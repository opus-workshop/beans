## Problem

`main.rs` is a 688-line match statement. Every new command adds to it, causing tight coupling between cli.rs, mod.rs, and main.rs (15 co-commits).

## Approach

Move command dispatch logic out of main.rs. Options:
1. **Move match into lib.rs** as `pub fn dispatch(cli: Cli) -> Result<()>` — main.rs becomes 10 lines
2. **Command trait** — each command implements a trait, dispatch is automatic

Option 1 is simpler and sufficient. main.rs should just parse args + call dispatch.

### Files to modify
- `src/main.rs` — reduce to parse + dispatch call
- `src/lib.rs` — add `pub fn dispatch()` with the match block

### Acceptance Criteria
- [ ] main.rs under 200 lines
- [ ] All commands still work
